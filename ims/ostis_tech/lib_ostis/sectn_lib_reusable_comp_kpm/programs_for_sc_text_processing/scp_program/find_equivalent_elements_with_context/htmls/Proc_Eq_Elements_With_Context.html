Задачей поиска общих элементов для двух заданных множеств является нахождение общих элементов для двух заданных множеств в рамках некоторого контекста. Первым и вторым входными параметрами данной scp-программы являются два анализируемых множества, третьим входным параметром является множество, куда будут занесены общие для двух множеств элементы. Четвертым входным параметром является контекст, в рамках которого будет осуществляться поиск.
