Задачей scp-программы поиска непосредственных связей между двумя понятиями с заданным контекстом является нахождение непосредственных связей между двумя понятиями в рамках некоторого контекста. Первым и вторым входными параметрами являются два анализируемых понятия, третьим входным параметром является узел ответа, куда будут занесены все удовлетворяющие поиску конструкции. Четвертым входным параметром является контекст, в рамках которого будет осуществляться поиск. В ходе выполнения программы явно указывается связь с ключевым узлом sc-отношение, позволяющим отличить абсолютное понятие от относительного..
