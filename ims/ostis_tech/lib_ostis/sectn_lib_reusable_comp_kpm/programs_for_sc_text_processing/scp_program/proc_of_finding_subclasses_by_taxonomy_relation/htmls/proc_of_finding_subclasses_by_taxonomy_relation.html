Задачей scp-программы поиска подклассов по отношениям таксономии является поиск подклассов по отношениям таксономии. Первым входным параметром является класс, для которого нужно находить подклассы. Вторым входным параметром является множество, куда будут занесены сам класс и все его подклассы под отношением таксономии.
