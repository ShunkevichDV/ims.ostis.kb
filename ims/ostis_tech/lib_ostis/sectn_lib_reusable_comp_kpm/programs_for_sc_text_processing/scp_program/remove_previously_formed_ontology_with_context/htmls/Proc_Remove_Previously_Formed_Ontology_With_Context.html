Задачей scp-программы удаления ранее сформированной онтологии является поиск и удаление из памяти структуры, являющейся онтологией заданного типа для заданной предметной области. Первым входным параметром является предметная область, для которой необходимо найти и удалить отнологию. Вторым входным паретром является класс онтологии, к которому принадлежит онтология, подлежащая удалению. Третьим входным параметром является контекст, в рамках которого будет осуществляться поиск онтологий предметной области.
