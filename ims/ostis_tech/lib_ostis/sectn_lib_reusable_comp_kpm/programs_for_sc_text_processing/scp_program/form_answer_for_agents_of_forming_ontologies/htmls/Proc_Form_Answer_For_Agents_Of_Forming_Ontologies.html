Задачей scp-программы формирования ответа для агентов формирования онтологий является добавление в ответ агента формирования онтологии всех элементов онтологии, знаков предметной области и онтологии, класса онтологии, а также обязательных ключевых узлов таких как предметная область и онтология* и передаваемых в качестве параметра.

Входные парамеры:
  1. сформированная онтология;
  2. предметная область, для которой осуществляется формирование онтологии;
  3. класс онтологии;
  4. множество ключевых узлов, которые необходимо добавить в ответ;
  5. ответ scp-прграммы.
