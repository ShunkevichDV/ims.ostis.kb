Задачей scp-программы поиска иллюстраций с заданным контекстом является поиск иллюстраций для заданного sc-элемента в рамках некоторого контекста. Первым входным параметром данной scp-программы является сам sc-элемент, для которого необходимо найти все иллюстрации, вторым входным параметром является узел ответа, куда будут занесены все найденные конструкции. Третьим входным параметром является контекст, в рамках которого будет осуществляться поиск. В ходе выполнения программы явно указывается связь со следующими ключевыми узлами: трансляцией sc-текста* и атрибутом пример'.
