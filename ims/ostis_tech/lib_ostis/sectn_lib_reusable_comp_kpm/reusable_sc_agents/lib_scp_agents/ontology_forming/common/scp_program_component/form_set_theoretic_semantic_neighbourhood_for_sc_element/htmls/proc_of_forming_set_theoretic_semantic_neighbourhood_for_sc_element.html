Задачей scp-программы формирования теоретико-множественной семантической окрестности sc-элемента является формирование теоретико-множественной семантической окрестности sc-элемента.

Первым входным параметром scp-программы является sc-элемент, исследуемый в рамках предметной области. Вторым входным параметром - знак теоретико-множественной семантической окрестности sc-элемента, для которого формируется теоретико-множественная семантическая окрестность.

В ходе программы явно указывается связь со следующими ключевыми узлами: ключевой sc-элемент и исследуемыми отношениями Предметной области множеств и Предметной области связей и отношений.
