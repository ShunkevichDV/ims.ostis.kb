Задачей агента поиска sc-элементов, имеющих системный идентификатор, но не имеющих ни одного основного является поиск sc-элементов, имеющих системный идентификатор, но не имеющих ни одного основного. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска sc-элементов, имеющих системный идентификатор, но ни одного основного. Первым аргументом запроса является раздел базы знаний, в рамках которого анализируются sc-элементы на предмет наличия у них системного идентификатора и, как минимум, одного основного идентификатора. При необходимости ограничения области поиска агента в качестве второго аргумента запроса подается контекст действия, в рамках которого будет осуществляться проверка.
