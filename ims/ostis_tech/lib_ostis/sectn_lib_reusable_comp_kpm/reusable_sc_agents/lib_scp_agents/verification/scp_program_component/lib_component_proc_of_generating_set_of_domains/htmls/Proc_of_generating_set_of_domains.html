Задачей scp-программы создания множества доменов является создание множества доменов. Первым входным параметром является знак множества, куда добавятся знаки классов. Вторым входным аргументом является знак отношения, домены которого будут добавлены во множество. Третьим входным аргументом является знак домена, который будет добавлен во множество. Четвертым входным аргументом является ролевое отношение, показывающее порядок следования доменов.
