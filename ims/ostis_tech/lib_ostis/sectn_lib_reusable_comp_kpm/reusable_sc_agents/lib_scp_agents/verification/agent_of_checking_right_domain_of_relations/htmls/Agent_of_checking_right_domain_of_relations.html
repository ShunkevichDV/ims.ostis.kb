Задачей агента проверки элементов, участвующих в отношении, на соответствие доменам для указанного отношения является проверка элементов, участвующих в отношении, на соответствие доменам для указанного отношения. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу проверки элементов, участвующих в отношении . Первым входным параметром является знак структуры, в рамках которой будут проверяться отношения на соответствие доменам. При необходимости ограничения области поиска агента в качестве второго аргумента запроса подается контекст действия, в рамках которого будет осуществляться проверка. В ходе выполнения агента явно указывается связь с scp-программой создания множества доменов, scp-программой генерации ответа для агентов верификации.
