Задачей scp-программы сортировки дерева соответствий является очистка дерева от пар соответствия, которые не связаны с конкретным фрагментом базы знаний. Первым входным параметром данной scp-программы является анализируемое дерево соответствий, полученное в результате работы оператора поиска по произвольному образцу, вторым - фрагмент базы знаний, элементы которого должны являться элементами пар соответствия итогового дерева. Данная scp-программа не имеет связи ни с одним из ключевых узлов базы знаний.
