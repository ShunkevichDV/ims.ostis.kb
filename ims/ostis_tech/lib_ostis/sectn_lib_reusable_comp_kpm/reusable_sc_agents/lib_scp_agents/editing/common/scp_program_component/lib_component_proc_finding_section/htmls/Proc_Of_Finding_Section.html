Задачей scp-программы поиска разделов является нахождение всех разделов, в которые входит указанный фрагмент базы знаний. Первым входным параметром данной scp-программы является множество, куда будут занесены все разделы, в которые входит заданный фрагмент базы знаний, вторым - анализируемый фрагмент базы знаний. Данная scp-программа не имеет связи ни с одним из ключевых узлов базы знаний.
