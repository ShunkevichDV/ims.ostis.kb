Задачей агента поиска утвержденных пользователем запросов является поиск утвержденных пользователем предложений. На вход данному агенту подается пользователь.Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска всех утверждённых конкретным пользователем предложений.При необходимости ограничения области поиска агента в качестве второго аргумента запроса подается контекст действия, в рамках которого будет осуществляться поиск. Результатом выполнения команды является вывод на экран всех утвержденных конкретным пользователем предложений.
