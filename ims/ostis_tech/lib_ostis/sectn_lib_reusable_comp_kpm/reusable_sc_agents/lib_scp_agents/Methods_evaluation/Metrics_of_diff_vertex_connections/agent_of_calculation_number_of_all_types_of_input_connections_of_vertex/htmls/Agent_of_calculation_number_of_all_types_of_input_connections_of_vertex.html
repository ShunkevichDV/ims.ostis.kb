Задачей агента подсчета среднего числа различных типов входящих связей вершины онтологии является подсчет среднего числа типов входящих связей для заданной вершины онтологии. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу подсчета среднего числа различных типов входящих связей вершины онтологии. Единственным аргументом запроса является знак онтологии. В ходе выполнения агента явно указывается связь с scp-программой вычисления различных типов входящих связей всех вершин онтологии, scp-программой подсчета среднего значения и scp-программой генерации ответа для агентов оценки качества. Для достижения цели находятся следующие узлы предметной области, онтологию которой подали на вход агенту:

  • Все исследуемые отношения' данной предметной области.
  • Все немаксимальный класс объектов исследования’ данной предметной области.
  • Максимальный класс объектов исследования’ данной предметной области.
Для подсчёта среднего числа различных типов входящих связей вершины онтологии количество всех найденных дуг, входящих в перечисленные выше вершины, делится на общее число данных вершин. Также в данном агенте предусмотренна верификация поданной на вход онтологии. Если аргумент не принадлежит ни одному подклассу класса онтология, то выводится сообщение о некорректности аргумента (Wrong first parameter!It must be ontology).
