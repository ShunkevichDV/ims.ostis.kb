Задачей агента вычисления количества вершин с разными типами входящих связей по отношению ко всем вершинам онтологии является вычисление количества вершин с разными типами входящих связей деленное на количество всех вершин онтологии. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу вычисления количества вершин с разными типами входящих связей по отношению ко всем вершинам онтологии. Единственным аргументом запроса является знак онтологии. В ходе выполнения агента явно указывается связь с scp-программой подсчета числа различных типов входящих связей вершин онтологии, scp-программой подсчета мощности множества и scp-программой генерации ответа для агентов оценки качества. Для достижения цели находятся следующие узлы предметной области, онтологию которой подали на вход агенту:

  • Все исследуемые отношения' данной предметной области.
  • Все немаксимальный класс объектов исследования’ данной предметной области.
  • Максимальный класс объектов исследования’ данной предметной области.
Результат данного агента характеризует качество поданной на вход онтологии. Качество определяется количеством "нежелательных" вершин среди всех вершин онтологии. Под "нежелательной" понимаются вершины, у которых больше одной входящей дуги. Чем меньше таких вершин, тем лучше качество онтологии. Также в данном агенте предусмотрена верификация поданной на вход онтологии. Если аргумент не принадлежит ни одному подклассу класса онтология, то выводится сообщение о некорректности аргумента (Wrong first parameter!It must be ontology).
