Задачей агента вычисления среднего числа различных типов исходящих связей вершины онтологии является вычисление среднего числа различных типов исходящих связей вершины онтологии. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу вычисления среднего числа различных типов исходящих связей вершины онтологии. Единственным аргументом запроса является знак онтологии. В ходе выполнения агента явно указывается связь с scp-программой подсчета числа различных типов исходящих связей вершин онтологии, scp-программой подсчета среднего значения и scp-программой генерации ответа для агентов оценки качества. Для достижения цели находятся следующие узлы предметной области, онтологию которой подали на вход агенту:

  • Все исследуемые отношения' данной предметной области.
  • Все немаксимальный класс объектов исследования’ данной предметной области.
  • Максимальный класс объектов исследования’ данной предметной области.
Для подсчёта среднего числа различных типов исходящих связей вершины онтологии количество всех найденных исходящих из перечисленных выше вершин дуг делится на общее число данных вершин.
