Задачей агента подсчёта количества знаков структур в струтуре является подсчет количества знаков структур в заданной структуре. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу посчитать количество знаков структур в структуре. Единственным входным аргументом является структура, для которой необходимо подсчитать количество знаков структур, принадлежащих ей. Вторым и необязательным параметром является контекст, в рамках которого необходимо осуществлять поиск.

    В ходе выполнения агента явно указывается связь со следующими scp-программами:
  • scp-программа генерации ответа для агентов оценки качества;
  • scp-программа добавления сообщения об ошибке;
  • scp-программа подсчёта мощности множества без учёта кратных вхождений;
  • scp-программа подсчёта мощности множества без учёта кратных вхождений в рамках заданного контекста.
