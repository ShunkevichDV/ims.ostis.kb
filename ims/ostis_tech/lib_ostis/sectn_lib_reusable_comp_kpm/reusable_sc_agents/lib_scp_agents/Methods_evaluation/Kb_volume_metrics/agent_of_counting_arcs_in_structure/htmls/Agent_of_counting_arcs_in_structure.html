Задачей агента подсчёта количества sc-дуг в струтуре является подсчёт количества sc-дуг в заданной структуре. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу посчитать количество sc-дуг в структуре. Входным аргументом является структура, для которой необходимо подсчитать количество sc-дуг. При необходимости ограничения области поиска агента в качестве второго аргумента запроса подается контекст действия, в рамках которого будет осуществляться поиск.

    В ходе выполнения агента явно указывается связь со следующими scp-программами:
  • scp-программа генерации ответа для агентов оценки качества;
  • scp-программа добавления сообщения об ошибке.
