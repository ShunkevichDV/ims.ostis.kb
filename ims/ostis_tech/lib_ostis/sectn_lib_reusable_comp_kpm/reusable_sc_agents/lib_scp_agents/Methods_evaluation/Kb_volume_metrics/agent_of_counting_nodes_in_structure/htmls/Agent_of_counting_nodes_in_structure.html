Задачей агента подсчёта количества sc-узлов в струтуре является подсчет количества sc-узлов в заданной структуре. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу посчитать количество sc-узлов в структуре. Единственным входным аргументом является структура, для которой необходимо подсчитать количество sc-узлов.

    В ходе выполнения агента явно указывается связь со следующими scp-программами:
  • scp-программа генерации ответа для агентов оценки качества;
  • scp-программа добавления сообщения об ошибке;
  • scp-программа подсчёта мощности множества без учёта кратных вхождений.
