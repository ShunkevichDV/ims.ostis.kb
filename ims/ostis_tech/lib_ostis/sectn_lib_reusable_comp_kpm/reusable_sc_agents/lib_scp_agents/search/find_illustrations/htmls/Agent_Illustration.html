Задачей агента поиска иллюстраций для заданного объекта является нахождение иллюстраций для заданного объекта. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска иллюстраций для заданного объекта. Аргументом запроса является объект, для которого необходимо найти изображения. При необходимости ограничения области поиска агента в качестве второго аргумента запроса подается контекст действия, в рамках которого будет осуществляться поиск.
