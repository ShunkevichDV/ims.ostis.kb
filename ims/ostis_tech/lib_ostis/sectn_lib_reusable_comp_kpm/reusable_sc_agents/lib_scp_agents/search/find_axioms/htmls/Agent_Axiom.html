Задачей агента поиска аксиом заданной sc-онтологии является нахождение аксиом заданной sc-онтологии. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска аксиом заданной sc-онтологии. Аргументом запроса является sc-онтология, в рамках которой необходимо найти все аксиомы. При необходимости ограничения области поиска агента в качестве второго аргумента запроса подается контекст действия, в рамках которого будет осуществляться поиск.
