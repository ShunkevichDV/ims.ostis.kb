Команда пользовательского интерфейса для поиска всех операторов scp-программы предназначена для поиска всех операторов scp-программы. Единственным аргументом класса действий является знак этой scp-программы. Результатом выполнения команды является множество, содержащее все операторы заданной scp-прогаммы.
