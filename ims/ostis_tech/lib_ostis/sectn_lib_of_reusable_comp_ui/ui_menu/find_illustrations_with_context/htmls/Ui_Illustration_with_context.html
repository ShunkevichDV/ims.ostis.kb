Команда поиска иллюстраций для заданного объекта с заданным контекстом предназначена для нахождения иллюстраций для заданного объекта в рамках контекста, который ограничивает область поиска агента. Первым аргументом команды является объект, для которого необходимо найти иллюстрации. Вторым аргументом является контекст действия, в рамках которого будет осуществляться поиск. Результатом выполнения запроса является множество изображений, найденных для объекта, сам объект и множество связывающих их дуг с отношением трансляция sc-текста* и атрибутом пример'.
