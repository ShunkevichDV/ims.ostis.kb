Команда пользовательского интерфейса для удаления точки останова предназначена для удаления точки останова из scp-программы. Первым аргументом запроса является знак scp-программы, вторым – знак scp-оператора, входящего в состав этой scp-программы. Результатом выполнения команды является удаление точки останова из scp-программы.
