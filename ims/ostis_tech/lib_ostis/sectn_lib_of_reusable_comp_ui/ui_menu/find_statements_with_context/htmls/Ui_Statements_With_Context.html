Команда поиска утверждений об объекте предназначена для нахождения утверждений об объекте в рамках контекста, который ограничивает область поиска агента. Первым аргументом команды является объект, о котором необходимо найти утверждения. Вторым аргументом является контекст действия, в рамках которого будет осуществляться поиск. Результатом выполнения запроса является множество утверждений, найденных для объекта, сам объект и множество связывающих их дуг с отношением трансляция sc-текста* и атрибутом пример', а также выборка утверждений, использующихся решателем и связанных с ключевым понятием отношением основные утверждения*.
