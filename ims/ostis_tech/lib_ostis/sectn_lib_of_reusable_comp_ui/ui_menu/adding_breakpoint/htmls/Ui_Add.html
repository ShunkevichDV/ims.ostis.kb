Команда пользовательского интерфейса для добавления точки останова предназначена для добавления точки останова в scp-программу. Первым аргументом запроса является знак scp-программы, вторым – знак scp-оператора, входящего в состав этой scp-программы. Результатом выполнения команды является добавление точки останова в scp-программу.
