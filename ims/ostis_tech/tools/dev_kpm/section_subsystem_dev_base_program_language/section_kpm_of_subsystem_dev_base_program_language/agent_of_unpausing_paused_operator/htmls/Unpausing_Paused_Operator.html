Задачей агента продолжения оператора является продолжение оператора scp-программы. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу продолжения оператора. Первым аргументом является знак scp-программы, второй – знак scp-оператора, входящего в состав этой scp-программы.
