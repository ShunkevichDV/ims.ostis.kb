Команда пользовательского интерфейса поиска всех точек останова в рамках scp-процесса предназначена для поиска всех точкек останова в рамках scp-процесса. Единственным аргументом класса действий, соответствующего данному агенту, является знак scp-процесса, с которым будет выполнено соответствующее действие. Результатом выполнения команды является множество, содержащее все точки останова заданной scp-прогаммы.
