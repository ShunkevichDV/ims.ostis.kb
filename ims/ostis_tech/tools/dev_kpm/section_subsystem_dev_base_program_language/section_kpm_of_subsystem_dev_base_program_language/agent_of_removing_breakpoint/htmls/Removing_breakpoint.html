Задачей агента удаления точки останова из scp-программы является удаление точки останова из scp-программы. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу удаления точки останова. Первым аргументом является знак scp-программы, второй – знак scp-оператора, входящего в состав этой scp-программы.
