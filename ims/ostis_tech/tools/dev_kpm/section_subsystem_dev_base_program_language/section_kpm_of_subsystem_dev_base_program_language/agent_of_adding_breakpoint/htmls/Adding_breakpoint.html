Задачей агента добавления точки останова в scp-программу является добавление точки останова в scp-программу. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу добавления точки останова. Первым аргументом является знак scp-программы, второй – знак scp-оператора, входящего в состав этой scp-программы.
