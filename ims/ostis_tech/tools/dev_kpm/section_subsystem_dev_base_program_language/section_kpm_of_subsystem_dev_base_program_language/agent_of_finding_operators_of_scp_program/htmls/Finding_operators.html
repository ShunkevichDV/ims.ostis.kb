Задачей агента поиска всех scp-операторов в рамках scp-программы является поиск всех операторов в рамках scp-программы. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска всех операторов scp-программы. Единственным аргументом класса действий является знак этой scp-программы.
