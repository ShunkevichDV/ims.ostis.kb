There is opportunity not to enter names for all sc-connectors in sc.s-sentences of 2 level.

To do this,sc.s separators are entered that are images of sc-connectors, as well as non-atomic IDs connectors type (...).

<* non-atomic ID of connector*> ::= ( <* the name of sc-element *> <* image of connector *> <* the name of sc-element *> [*suffix *]

Possible images of sc-connectors is given in the section on sc.s-separators.

If there are multiple connectors between some sc-elements,suffixes are introduced to avoid ambiguity, for example (v1 -> v2)1, (v1 -> v2)2.

The use of non-atomic identifiers of the form <...>, {...}, [...] is not allowed.
