Aлгол (англ. ALGOL от англ. algorithmic — алгоритмический и англ. language — язык), название семейства императивных языков программирования, применяемых при составлении программ для решения научно-технических задач на ЭВМ.

Алгол был первым языком программирования, использующим пары begin end для выделения блоков выражений.
