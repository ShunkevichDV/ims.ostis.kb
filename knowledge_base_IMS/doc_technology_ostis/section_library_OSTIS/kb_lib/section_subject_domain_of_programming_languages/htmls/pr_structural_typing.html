Структурная типизация определяет совместимость типов данных, используя их структуры. Два типа данных считаются эквивалентными, если они имеют идентичную структуру, т.е. для каждого элемента (метода или атрибута) определения одного типа существует идентичный элемент определения второго типа, и наоборот.

Структурная типизация является противоположностью номинативной типизации.
