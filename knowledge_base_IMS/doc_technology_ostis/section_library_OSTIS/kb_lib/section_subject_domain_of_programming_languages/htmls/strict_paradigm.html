В строгом языке программирования могут быть определены только строгие функции.

В денотационной семантике языков программирования функция f является строгой если f (_|_) = |. Сущность «_|_» обозначает выражение, которое не возвращает нормального значения по причине выполнения бесконечного цикла или прекращения из-за ошибки (к примеру, деления на ноль).
